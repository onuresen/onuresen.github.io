ONEXUS
A graph viewer that Claude can query and control live.

Browser-based graph visualization and composition tool built for the AEC industry. Runs as plain HTML with no build step — load any JSON graph, IFC file, or CSV and explore relationships visually. Connects to Claude via an MCP server so an AI agent can navigate, highlight, and filter the graph in real time.

What's different
Ask Claude: "Show me all nodes connected to Room 105" — ONEXUS highlights them live. The same graph you see in the browser is the one the AI is navigating.

Thinking Hub
20+ tools. One browser tab. No install.

A personal productivity suite — project tracker, decision log, KMQT board, goal planner, Pomodoro timer, risk register, and 15+ more tools — all running inside a single shell HTML file with no server and no build step. Data syncs to the cloud via Supabase or stays local. Connects to an Obsidian vault through the File System Access API.

What's different
Every tool shares the same data store and can deep-link to each other. A decision in Decision Hub appears as context in the project tracker. Zero friction — open the file, start working.
Vanilla JS No build step Supabase localStorage File System API Obsidian sync

Kit of Parts
A 3D configurator that takes Industrialized Construction seriously.

Interactive 3D building configurator for modular and prefabricated systems. Parts carry cost, weight, carbon, and compliance data. Run earthquake shake simulations, fire spread visualizations, crane collision checks, and sustainability metrics — all in the browser. Built against real Japanese standards: seismic grades 1/2/3, BSL fire resistance, CASBEE, Ken grid.

What's different
Not a geometry viewer — a decision tool. Every part has variants with trade-off data. The AI carbon optimiser swaps your assembly toward the lowest embodied carbon configuration automatically.
